# /sub

### 解释

查询所有通过/new指令保存的订阅名，加上对应参数可以输出订阅链接。

### 参数

无参数：

```
/sub
```

有参数：

```
/sub <订阅名> <访问密码>
```

```
<订阅名>    这个订阅的别名，通过/new设置的
<访问密码>    (可选)，该项不填将不设置访问密码，访问密码的意义其实并不大，除非当前TG号有多人共用。

```

### 使用

> ```
> /sub 谷歌云
> ```
>
> 有访问密码：
>
> ```
> /test 谷歌云 123456
> ```
>
> 无参数：
>
> ```
> /sub
> ```

### 特性

无参数将输出所有订阅名



管理员身份调用时，可以查看所有订阅名，并且可以拿到所有订阅链接，不论是否是订阅的所有者。

用户身份时，可以查看自己添加的订阅，可以拿取自己的订阅，无法以任何方式获取并展示其他人的订阅。
